# Expense Tracker

**Автор:** Иванов Александр Владимирович

## Описание
Приложение для учёта личных расходов с графическим интерфейсом на Python (tkinter). Позволяет добавлять траты, фильтровать по категориям и датам, считать сумму за период, сохраняет данные в JSON.

## Возможности
- Добавление расходов (сумма, категория, дата)
- Проверка корректности ввода (положительная сумма, формат даты)
- Фильтрация по категории и диапазону дат
- Подсчёт суммы за выбранный период
- Удаление записей
- Автосохранение/загрузка из JSON

## Как использовать
1. Установите Python 3.6+
2. Запустите `python expense_tracker.py`
3. Для добавления расхода заполните форму и нажмите "Добавить расход"
4. Для фильтрации выберите категорию и/или укажите период и нажмите "Применить фильтр"
5. Для подсчёта суммы за период укажите даты и нажмите "Рассчитать сумму..."

## Примеры использования
**Добавление расхода:**
- Сумма: 350.50
- Категория: Еда
- Дата: 2026-04-28

**Фильтр по категории** → показывает только выбранную категорию

**Подсчёт суммы за март 2026:**  
от 2026-03-01 до 2026-03-31 → выведет итоговую сумму
