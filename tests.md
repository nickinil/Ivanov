# Тестирование приложения Expense Tracker

**Проект:** Expense Tracker  
**Автор:** Иванов Александр Владимирович  
**Назначение тестирования:** проверка корректной работы GUI-приложения для учёта личных расходов.

---

## 1. Проверяемые функции

В ходе тестирования проверялись следующие возможности программы:

- запуск приложения;
- добавление нового расхода;
- проверка корректности суммы;
- проверка корректности даты;
- сохранение данных в JSON;
- загрузка данных из JSON;
- фильтрация расходов по категории;
- фильтрация расходов по диапазону дат;
- подсчёт суммы за выбранный период;
- удаление выбранной записи.

| № | Действие | Входные данные | Ожидаемый результат | Результат |

## 5. Проверка JSON

Пример структуры данных в JSON-файле:

```json
[
  {
    "id": 1,
    "amount": 350.5,
    "category": "Еда",
    "date": "2026-04-28"
  }
]
```

Проверено:

- данные сохраняются в формате JSON;
- русские категории отображаются корректно;
- при перезапуске программы данные загружаются обратно;
- после удаления записи JSON обновляется.
